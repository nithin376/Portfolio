1st project

GYM WEBSITE

A comprehensive gym web application built from the ground up using the PERN stack (PostgreSQL, Express.js, React.js, Node.js). Users can browse facilities, compute BMI with tailored feedback, and enroll in membership plans—all in a secure and intuitive environment. Demonstrates expertise in building interactive, data-driven full stack applications.

Stack Used:

  • PostgreSQL for data storage
  • Express and Node.js for backend integration
  • React.js for frontend

2nd project

Rock Paper Scissor Game

This project is a classic Rock Paper Scissors game implemented with an interactive graphical user interface. Users can play against the computer, which makes random choices each round. The game provides real-time visual feedback by displaying images representing the user’s and computer’s selections. It also shows the game outcome instantly, whether the player wins, loses, or draws in each round. The application features buttons for player choices and a reset option to start a new game, creating a smooth and engaging user experience.

Stack Used:

  • HTML and CSS for structuring and designing the website
  • JavaScript to make the website interactive
